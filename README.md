# 忆伴相册 Yiban Memory Frame

面向独居、空巢及轻度认知衰退老人的 AI 亲情相册陪伴终端。

忆伴相册不是机器人、不是数字人，也不是一台带摄像头的看护设备。它以家庭照片和生活回忆为核心载体，通过自然语音、长期记忆、亲情留言和 AI 摘要，陪老人回忆过去、表达当下，并把这些珍贵片段重新连接给家人。

## 核心理念

我们不希望 AI 替代亲情，而是让 AI 成为家庭情感的转译者。

老人面对相册聊天、看照片、讲过去时，系统会理解照片中的人物、场景和家庭关系，接住老人的话，温和引导回忆，并把适合分享的内容整理成亲情摘要推送给子女。子女看到的不再只是照护提醒，而是父母今天想起了什么、牵挂了谁、有什么值得回应的生活片段。

> 用技术建立连接，而不是增加隔阂。

## 产品形态

- 电子相册 / 智能相框形态
- 无摄像头，降低监控感和隐私压力
- 不使用拟人数字人、虚拟主播或机器人脸
- 平时自动轮播家庭照片、回忆、留言和提醒
- 以语音为主要交互方式，屏幕承担照片、文字、状态和记忆呈现
- 可加入人体感应、麦克风阵列、物理麦克风关闭键和柔和状态灯

## 核心功能

1. AI 相册理解  
   扫描家庭相册，识别照片内容、人物关系、时间线和场景线索，建立可持续增长的家庭记忆库。

2. 面对面陪伴模式  
   老人坐到相册前时，设备进入可见的短时交谈状态，可以直接说话、连续交流、随时打断，无需反复喊唤醒词。

3. 唤醒名称交互  
   老人不在设备面前时，可以通过唤醒名称远场呼叫设备。

4. 回忆陪伴  
   AI 围绕照片、旧事和老人表达进行陪伴，不泛泛闲聊，而是接上老人的真实生活记忆。

5. 双向亲情留言  
   子女可以发送照片、文字和语音留言；老人可以自然语音回复；AI 可帮助老人整理成清楚、温和的留言。

6. AI 亲情摘要  
   系统把老人日常聊天中的回忆、情绪和牵挂整理成适合子女阅读的摘要，促进子女回应。

7. 亲情声音模式  
   默认使用系统声音。仅在老人主动请求，并且家属授权后，才使用家属声音风格朗读真实留言或陪伴回忆，不冒充家属本人。
